5.
    5 просветителей есть изначально.
    Каждый час один просветитель обращает двух человек.
    Каждый час один из просветителей разубеждается в своих взглядах и снимает шапочку из фольги, при этом уже после того, как он обратил двух других.
    тогда пусть f(n) - количество обращенных по прошествии n часов. 
    
    Ясно, что f(0) = 5.
    	      f(1) = 5 +5*2 -1 = 14
    Составим общую формулу.
    
    По прошествии n часов сообщество просветителей будет состоять из: сообщества просветителей, которое было сформировано по прошествии n-1 часов. Также в него будет входить сообщество новообращенных, состоящее из 2 * f(n-1) человек (ведь каждый "старик" обратит двух новых), но 1 перестанет носить шапочку из фольги. Значит формула примет вид:
    f(n) = f(n-1) + 2f(n-1) - 1 = 3f(n-1) -1
    
 6.
    Составить рекуррентное соотношение описывающее количество бинарных векторов размера n таких, что:
    			     (x1<=x2>=x3<=x4)
    Пусть f(n) описывает ответ на задачу
    f(0) = 1
    f(1) = 2
    f(n,x) - ответ на задачу, если n-й элемент равен x
    Разберем случаи:
    	             1) n - чётное.
    	                Тогда вектор примет вид:  ...>=xn-3<=xn-2>=xn-1<=xn 
    	                Здесь снова рассмотрим два случая: 1) xn = 1
    	                                                      В таком случае никаких ограничений на n-1-й элемент не накладывается(он может быть как 0 так и 1), значит в таком случае количество f(n,1) = f(n-1)
    	                                                   2) xn = 0
    	                                                      В таком случае n-1-й элемент может быть равен только 0, но не будет ограничений на n-2й, таким образом f(n,0) = f(n-1,1) = f(n-2)
    	              Итого имеем f(n) = f(n-1) + f(n-2)
    	             2) n - нечётное
    	                Тогда вектор примет вид:  ...<=xn-3>=xn-2<=xn-1>=xn
    	                	Снова рассмотрим 2 случая:
    	                	                           1) xn = 1.
    	                	                              Тогда у нас предыдущий элемент определяется однозначно, а а предпредыдущий свободен, значит f(n,1) = f(n-2)
    	                	                           2) xn=0, тогда свободен предыдущий элемент, то есть f(n,0) = f(n-1)
    	                Итого имеем f(n) = f(n-1) + f(n-2)
    Как мы видим ответ не зависит от четности n и равен f(n) = f(n-1) + f(n-2)
7. 
    Дано множество натуральных чисел не больших n - {1,2,...,n}. Составить рс отражающее:
    	а) количество таких подмножеств, не содержащих двух последовательных натуральных чисел.
    	   Пусть  f(n)  - ответ на задачу.
    	   f(0) = 1
    	   f(1) = 2
    	   
    	   f(n) - ?
    	   
    	   Покрасим последний элемент в синий цвет, а предпоследний в красный.
    	   Тогда совершенно ясно, что синий и красный элемент не могут быт в одном подмножестве
    	   Тогда f(n) будет состоять из - подмножеств без синего и без красного элемента, подмножеств с синим, подмножеств с красным.
    	   Ясно, что число подмножеств без синего и без красного элемента - f(n-2) ({1...n-2, синий, красный} - выкинем синий и красный)
    	   Число подмножеств с синим - f(n-1) - f(n-2)  (f(n-1) - все подмн-ва с синим и без него мн-ва {1...синий}, тогда вычтем все те, что без него)
    	   Число подмножеств с красным - f(n-2)  (то есть мы взяли все подмн-ва без синего и без красного и к каждому из них добавили красный)
    	   *интересно, что если определять число синих по аналогии с числом красных как f(n-3), то мы получим ответ f(n) = 2f(n-2) + f(n-3), который все еще правильный)
    	   Тогда сложив их получим: f(n) = f(n-1) + f(n-2)
    	б) Нет трех подряд идущих последовательных чисел
    	   Пусть f(n) - ответ на задачу
    	   
    	   f(0) = 1
    	   f(1) = 2
    	   f(2) = 4
    	   
    	   "Покрасим последний элемент в синий цвет, предпоследний в красный, а пред-предпоследний в зеленый
    	   
    	   Наш ответ будет состоять из всех подмн-в: не содержащих цветных элементов, содержащих максимум зеленый, максимум красный, максимум синий.
    	   ....,n-4,n-3 зеленый красный синий
    	   
    	   
    	    не содержащих цветных элементов - f(n-3)
    	    
    	    содержащих максимум зеленый - Возьмем  f(n-2) и вычтем все его ответы не содержащие зеленый то есть вычтем f(n-3)
    	    
    	    максимум красный - по аналогии с предыдущим имеем f(n-1) - f(n-2)
    	    
    	    Максимум синий - будем докладывать синий элемент в другие ответы. Возьмем все ответы не содержащие синий, то есть f(n-1), но тогда нам нужно учесть, что когда мы доложим к ним синий у нас образуется последовательность из зеленого синего и красного. Тогда вычте наборы содержащие красный и зеленый. Для этого будем докладывать зеленый и красный в другие множества. Ясно, что нам нужно рассматриать f(n-4), так как  n-3й элемнт не должен попать к зеленому и красному. Итого имеем f(n-1) - f(n-4) 
    	   
    	  
    	   Тогда это можно выразаить как f(n-3) + (f(n-2) - f(n-3)) + (f(n-1) - f(n-2)) + (f(n-1) - f(n-4)) = 2*f(n-1) - f(n-4) 
    	   
    	   
    	    Другой способ решения:
    	    
    	    Разобьем ответ на следующие: без цветных, только с синим, только с красным, только с зеленым, с синим и зеленым, красным и зеленым, синим и красным.
    	    
    	    Тогда все наборы не содержащие синий описываются как - f(n-1)
    	     
    	    Все наборы содержащие синий, но без красного - f(n-2)
    	    
    	    Синий с красным - f(n-3)
    	    
    	    Тогда ответ будет числами трибоначчи
    	    
    	    Важное отличие между двумя ответами - их степень

    марки по 5,10,15,20 сантиков.
    
    f(n) - ответ на задачу, где  n - бюджет

9. Сколькими способами можно создать n-полосный флаг. Полоса может быть - красной, зеленой, белой. Две красные полосы подряд не могут идти.
    f(n) - ответ на задачу
    Предположим, что мы раскрасили n-ю полосу. Мы могли покрасить её в белый, красный, зеленый. Белый и зеленый случаи не накладывают на нас никаких ограничений на дальнейшую раскраску полос. Покрасив в красный, следующая полоса может быть только белой или зеленой. Тогда переходя к следующей полосе, имеем белый и зеленый случаи, которые не накладывают ограничений на n-2ю полосу.
    тогда ответ примет вид f(n) = 2*(f(n-1) + f(n-2))
    В целом идея задачи идентична задаче про бинарные вектора

10.
    Сколько существует числовых последовательностей размера  n  составленных из 0,1,2,3 с четным количеством нулей.
    
    f(n) - ответ на задачу.
    
    Заметим, что общее количество искомых последовательностей - 4^n, а значит количество последовательностей длины n с нечетным числом нулей можно определить как : 4^n - f(n)
    
     Представим, что ставим некое число в n-й элемент пустой последовательности. Мы можем поставить туда ненулевой, тогда мы не накладываем никаких ограничений на остаток последовательности и просто обращаемся к f(n-1). Если же мы ставим в n-ю позицию ноль, то тогда нам надо число пос-тей длины n-1 с нечетным количеством единиц, то есть 4^(n-1) - f(n-1) итого наш ответ:
     f(n) = 4^(n-1) - f(n-1) + 3f(n-1) = 4^(n-1) + 2*f(n-1)
     
11.
    Сколько существует последовательностей длины n составленых из 0,1,2 в которых нет ни двух единиц, ни двух двоек подряд.
    
    f(n) - ответ на задачу.
    
    Представим, что мы ставим некий элемент в n-ю позицию пустой последовательности. Если мы ставим 0, то никаких ограничений на остаток не накладывается. Для визуализации того, что будет происходить при подстановке  последнего элемента построим дерево.
    				
    	n	0 0		0 1 2
    	n-1	0 1 2 0 1 2	0 1 2 0 2 0 1 
    				
    	Заметим, что подстановку единицы или двойки, можно представить как подстановку нуля.
    	0 0		0 1 2
    	0 1 2 0 1 2	0 1 2 0 2 0 1 
    	
    	Не хватает лишь еще одного нуля. По сути, этот ноль, можно представить как то, что мы начинаем новую последовательность.
    	Итого мы можем свести исходные варианты подстановки к двум подстановкам нуля и началом новой пос-ти размера n-1, но так как ее первый элемент фиксирован, это можно представить как просто последовательности размера n-2.
    	Итого наш ответ - 2*f(n-1) + f(n-2)
